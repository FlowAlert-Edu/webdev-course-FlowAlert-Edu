# Fundamentos de Desarrollo Web con HTML y CSS

## Descripción
Este repositorio contiene el material utilizado en el curso **Fundamentos de Desarrollo Web con HTML y CSS**, diseñado para estudiantes de secundaria entre 12 y 17 años sin experiencia previa en programación.

El curso enseña los conceptos básicos de HTML y CSS mediante ejemplos prácticos desarrollados en herramientas en línea, permitiendo a los estudiantes crear una página web de perfil personal sin necesidad de instalar software.

---

## Información General
- **Duración total del curso:** ~60 minutos
- **Público objetivo:** Estudiantes de 12 a 17 años
- **Prerrequisitos:** Ninguno
- **Herramientas utilizadas:**
  - Navegador Web (Chrome, Firefox, Edge o Safari)
  - CodePen
  - GitHub
  - YouTube
